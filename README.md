# 🚀 Interactive Username Validator

A modern and interactive username input component built using **HTML, CSS, and JavaScript**.
This project demonstrates real-time input handling, validation, and a visually appealing UI design.

---

## ✨ Features

* 🔤 Real-time uppercase conversion
* 🎯 Input guidance using **focus event**
* ⚠️ Validation message using **blur event**
* 🎨 Modern mesh gradient background
* 🧊 Glass-style UI (frosted card effect)
* ⚡ Smooth hover and animation effects

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**

  * Flexbox
  * Gradients (Mesh / Radial)
  * Transitions & Animations
* **JavaScript (Vanilla JS)**

  * DOM Manipulation
  * Event Handling (`focus`, `blur`, `keyup`)

---

## 🧠 Concepts Covered

* DOM (Document Object Model)
* Event-driven programming
* Client-side form validation
* Real-time data transformation
* UI/UX enhancement with CSS
